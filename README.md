# Diwali-sales-analysis
Analyzed Diwali sales data using Python in Jupyter Notebook to uncover customer purchasing trends and sales 
patterns
Performed data cleaning and manipulation using pandas to prepare the dataset for analysis.
Created insightful visualizations using matplotlib and seaborn to highlight key trends, top-performing categories, and 
customer behavior.
STRENGTHS
Derived meaningful insights to support business decisions and improve sales strategies during festive seasons.
